Praktek PemrogramanWeb

Sesi 1

Praktek Pemrograman Wab 1 akan membahas tentang cara menampilkan halaman web dengan mengunakan HTML, CSS dan Javascript. Mahasswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pemahaman tersebut kemudian menuliskan hasil kreasiya sendiri

Pada Sesi 1 mahasiswa mengenali dulu struktur dasar HTM beserta sintaks dasarnya

  • HTML
  • CSS
  • Javascript

Pada sesi 1 mahasiswa mengenali dulu struktur dasar HTML beserta sintaks dasarnya

  • Pengenalan HTML
  • Tabel
  • Form
